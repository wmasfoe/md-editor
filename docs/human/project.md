# 背景

此刻，我用 Typora 编辑器写下了这篇文档。

Typora 是一个很好的 Markdown 编辑器，所见即所得，给了我酣畅淋漓的 Markdown 体验。UI 也非常精美，且支持自定义主题、自定义图片位置、支持图床，LaTex 数学公式也支持。最重要的是在我的学生时代一直是一款免费的 Markdown 编辑器，虽然后续 1.0 版本收费了，但那时候我已经工作，立马购入了正版。

但是在我看来一直有一些痛点，比如==不支持 MDX 自定义组件==、不支持 AI 纠错、AI 生成文本。

最重要的就是不支持 MDX 组件，并且最近 AI 很火热啊，各种平台、APP 都恨不得接入一个 AI 功能。故决定实现一个 Typora like 的 Markdown 编辑器，并加入一些 AI 功能。
